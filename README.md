# ContainerTopology
Containerlab Docker based virtualized environment to practice routing and switching concepts. 
  - Utilized the sandbox nodes to troubleshoot potential configuration and/or data plane issues. 
  - Configured complete IGP, RSVP-TE core label switching, L2 and L3 VPN services, and eBGP/iBGP sessions.
  - Validated routing concept expectations and Nokia 7750 known issues.
